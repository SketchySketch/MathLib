Package function.line

Overview

The line package includes:

  • Point Declaration for points
  • LineDeclaration Declaration for lines
  • Line More utilities for lines
  • Calc.kt More calculations

LineDeclaration

Constructs a basic line.

It's not recommended to use this class, for Line is a better implementation.
